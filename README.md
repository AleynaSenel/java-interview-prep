# ☕ Java Interview & Exam Prep Bootcamp

Bu depo, kurumsal şirketlerin teknik mülakatlarına ve yetenek testlerine hazırlık sürecimi dokümante etmek amacıyla oluşturulmuştur. 6 gün sürecek olan bu yoğunlaştırılmış kamp boyunca, Java'nın temellerinden modern özelliklerine kadar kritik konuları teorik olarak özetleyip, pratik "lab" çalışmalarıyla kod üzerinde test ediyorum.

## 🎯 Amacım
Sadece teorik bilgi çalışmak yerine; kod okuma, hata bulma (debugging), çıktı tahmini yapma ve zihinde derleme (mental execution) pratikleri yaparak gerçek sınav koşullarına hazırlanmak. Bu depo, öğrendiğim bilgileri somut bir çıktıya dönüştürme ve uygulamalı öğrenme prensibimin bir parçasıdır.

## 🗓️ Kamp Programı (6 Günlük Plan)

* **Day 01: Fundamentals & Traps** - Veri tipleri, Type Casting, String Pool, Immutability, Döngü tuzakları.
* **Day 02: Object-Oriented Programming (OOP)** - Encapsulation, Inheritance, Polymorphism, Abstract Class vs Interface, `static`, `final`, `this`, `super`.
* **Day 03: Data Structures & Collections** - List, Set, Map farklılıkları, performans analizleri ve Array manipülasyonları.
* **Day 04: Exception Handling & Memory** - Try-catch-finally mekanizması, Checked/Unchecked exceptions, temel Garbage Collector mantığı ve Debugging pratikleri.
* **Day 05: Modern Java (Java 8+)** - Lambda Expressions ve Stream API (filter, map, collect).
* **Day 06: Final Mock Exam** - Zaman kısıtlamalı, karışık ve dikkat ölçen sınav simülasyonu çözümleri.

## 📂 Klasör Yapısı
Her günün notları ve o güne ait pratik lab kodları, ilgili günün klasörü içerisinde (`Day-01-Fundamentals` vb.) bulunabilir. Her klasörde konunun mülakatlarda en çok sorulan "hap bilgileri" ve zihinden derleyerek çözdüğüm örnek sorular yer almaktadır.
